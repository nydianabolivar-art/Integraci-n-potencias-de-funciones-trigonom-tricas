// módulo 00 — introducción

¿Por qué importa esto en ingeniería?

Las señales, los ciclos de producción, los datos periódicos y los sistemas operativos comparten una base matemática: las funciones trigonométricas. Integrarlas es calcular acumulaciones.

🎮
Ingeniería Multimedia
En audio y síntesis de sonido, una onda de audio es una suma de senos y cosenos (Serie de Fourier). La energía total de una señal en un intervalo se calcula integrando \(\sin^2\) y \(\cos^2\). En gráficos 3D, las normales de superficies esféricas involucran integrales de potencias trigonométricas. En animación paramétrica, las trayectorias circulares y las curvas de cámara usan exactamente estos integrales.
⚙️
Ingeniería Industrial
La producción en turnos, el mantenimiento cíclico de maquinaria y el consumo energético de plantas industriales siguen patrones periódicos. El costo marginal oscilatorio del proceso productivo se modela con \(\sin^2\) y \(\cos^2\). Calcular el costo total de un lote o el promedio de largo plazo requiere resolver estas integrales.
📊
Ciencia de Datos
Los kernels trigonométricos en machine learning (RBF, kernel de Mercer) y las transformaciones de Fourier discretas para análisis de series de tiempo requieren manejar potencias de funciones trigonométricas. El preprocesamiento de datos estacionales y la extracción de características periódicas en señales EEG, ECG o de mercado financiero involucran estos cálculos.
💻
Ingeniería de Software
En desarrollo de videojuegos y simulaciones físicas, la energía cinética rotacional, los cálculos de iluminación (BRDF — función de distribución de reflectancia bidireccional) y los shaders de gráficos involucran integrales trigonométricas. También aparecen en algoritmos de interpolación para animaciones y en el procesamiento de señales de sensores IoT.
🔗
Ingeniería de Sistemas
En telecomunicaciones y análisis de ancho de banda, la potencia de una señal se calcula con \(\int\sin^2(\omega t)\,dt\). En redes y teoría de colas, los patrones de tráfico son periódicos. El análisis de capacidad de sistemas y el dimensionamiento de servidores usando modelos de carga cíclica requieren resolver estas integrales.

Identidades Trigonométricas Esenciales

Tu "librería de funciones" para resolver integrales. Sin estas identidades, los métodos no funcionan.

💡
Principio clave: El objetivo es siempre reducir el exponente o cambiar la función a una forma que podamos integrar directamente. Las identidades son el mecanismo de transformación.
IdentidadFórmulaCuándo usarlaCarrera
Pitagórica base \(\sin^2x + \cos^2x = 1\) Reemplazar sin² o cos² cuando hay potencias mixtas todas
Reducción seno² \(\sin^2x = \dfrac{1-\cos 2x}{2}\) Potencias pares de seno. Fundamental en análisis de señales multimedia sistemas
Reducción coseno² \(\cos^2x = \dfrac{1+\cos 2x}{2}\) Potencias pares de coseno. Energía de señal, BRDF software datos
Tangente-secante \(\tan^2x = \sec^2x - 1\) Integrales de tan^n, optimización no lineal industrial
Producto sen·cos \(\sin x\cos x = \dfrac{\sin 2x}{2}\) Factorizar productos. Series de Fourier cruzadas datos multimedia
Producto sin²·cos² \(\sin^2x\cos^2x = \dfrac{1-\cos 4x}{8}\) Combinar ambas reducciones: caso más frecuente industrial sistemas
// truco mnemotécnico para las identidades de reducción
\[\text{función}^2(x) = \frac{1 \;\color{#f43f5e}{-}{\color{white}{/}}\color{#22c55e}{+}\; \cos(2x)}{2}\]

El signo es negativo (−) para seno y positivo (+) para coseno.
En código: sin² → (1 - cos(2x)) / 2  |  cos² → (1 + cos(2x)) / 2

\[\int\sin^m x\cos^n x\,dx \xrightarrow{m\text{ impar}} \int\sin^{m-1}x\cos^n x\underbrace{\sin x\,dx}_{-du}\]
// separar
Escribe \(\sin^m x = \sin^{m-1}x \cdot \sin x\) (m−1 es par)
// transformar
Usa \(\sin^{m-1}x = (1-\cos^2x)^{(m-1)/2}\). Todo queda en cos(x).
// sustituir
Sea \(u=\cos x\), \(du=-\sin x\,dx\). La integral se convierte en un polinomio en u.
// integrar
Aplica \(\int u^k\,du = u^{k+1}/(k+1)\) a cada término.
// revertir
Sustituye u → cos(x).
Ejemplo: \(\int\sin^3 x\,dx = \int(1-\cos^2x)\sin x\,dx \xrightarrow{u=\cos x} -\int(1-u^2)\,du = -u+\dfrac{u^3}{3}+C = -\cos x+\dfrac{\cos^3 x}{3}+C\)
// MÉTODO 3 — ambos pares → identidades de ángulo doble

Cuando m y n son ambos pares, no se puede "separar un factor". Se reduce el exponente usando las identidades de mitad:

\[\sin^4 x = \left(\frac{1-\cos 2x}{2}\right)^2 = \frac{1-2\cos 2x+\cos^2 2x}{4}\] \[= \frac{1}{4}\left(1-2\cos 2x+\frac{1+\cos 4x}{2}\right) = \frac{3}{8}-\frac{\cos 2x}{2}+\frac{\cos 4x}{8}\]
⚠️
Si después de reducir todavía aparece cos²(kx), aplica la identidad de nuevo. El proceso es recursivo hasta que todos los términos tengan exponente 1.
// MÉTODO 4 — reducción de índice para tangente
\[\int\tan^n x\,dx = \int\tan^{n-2}x\underbrace{(\sec^2 x-1)}_{\tan^2 x}\,dx = \frac{\tan^{n-1}x}{n-1}-\int\tan^{n-2}x\,dx\]

Casos base: ∫tan x dx = ln|sec x| + C  ·  ∫1 dx = x + C

Cada aplicación reduce el exponente en 2: tan⁴ → tan² → tan⁰ = 1. Igual que la recursión en programación.

Básico · 01 ∫ cos³(x) dx — potencia impar de coseno ›
Calcular: \(\displaystyle\int\cos^3 x\,dx\)
// solución paso a paso
separar un factor cos(x)
\(\int\cos^2 x\cdot\cos x\,dx\)
usar identidad pitagórica
\(\cos^2x=1-\sin^2x \;\Rightarrow\; \int(1-\sin^2x)\cos x\,dx\)
sustituir u = sin(x)
\(du=\cos x\,dx \;\Rightarrow\; \int(1-u^2)\,du\)
integrar
\(u - \dfrac{u^3}{3}+C\)
regresar a x
Sustituir u = sin(x)
\[\int\cos^3 x\,dx = \sin x - \dfrac{\sin^3 x}{3} + C\]
Básico · 02 ∫ sin²(x) dx — potencia par, ángulo doble ›
Calcular: \(\displaystyle\int\sin^2 x\,dx\)
// solución
identidad de reducción
\(\sin^2x=\dfrac{1-\cos(2x)}{2}\)
separar la integral
\(\dfrac{1}{2}\int 1\,dx - \dfrac{1}{2}\int\cos(2x)\,dx\)
integrar cada término
\(\dfrac{x}{2} - \dfrac{\sin(2x)}{4}+C\)
\[\int\sin^2 x\,dx = \dfrac{x}{2} - \dfrac{\sin(2x)}{4}+C\]
💡
Este es el resultado clave para calcular la potencia promedio de una señal sinusoidal — fundamental en multimedia y telecomunicaciones.
Intermedio · 03 ∫ sin³(x) cos²(x) dx — potencias mixtas ›
Calcular: \(\displaystyle\int\sin^3x\cos^2x\,dx\)
// estrategia: m=3 (impar) → Método 1
separar factor sin(x)
\(\int\sin^2x\cos^2x\cdot\sin x\,dx\)
reemplazar sin²x = 1 − cos²x
\(\int(1-\cos^2x)\cos^2x\sin x\,dx\)
expandir el producto
\(\int(\cos^2x-\cos^4x)\sin x\,dx\)
u=cos(x), du=−sin(x)dx
\(-\int(u^2-u^4)\,du = -\dfrac{u^3}{3}+\dfrac{u^5}{5}+C\)
sustituir u = cos(x)
Resultado final
\[\int\sin^3x\cos^2x\,dx = -\dfrac{\cos^3x}{3}+\dfrac{\cos^5x}{5}+C\]
Intermedio · 04 ∫₀^(π/2) sin²(x)cos²(x) dx — integral definida ›
Calcular: \(\displaystyle\int_0^{\pi/2}\sin^2x\cos^2x\,dx\)
// ambas potencias pares → identidad de producto
identidad de producto
\(\sin^2x\cos^2x = \dfrac{\sin^2(2x)}{4}\)
reducir sin²(2x)
\(\dfrac{\sin^2(2x)}{4}=\dfrac{1-\cos(4x)}{8}\)
integrar
\(\int_0^{\pi/2}\dfrac{1-\cos(4x)}{8}\,dx = \dfrac{1}{8}\left[x-\dfrac{\sin(4x)}{4}\right]_0^{\pi/2}\)
evaluar límites
sin(2π)=sin(0)=0 → \(\dfrac{1}{8}\cdot\dfrac{\pi}{2}=\dfrac{\pi}{16}\)
\[\int_0^{\pi/2}\sin^2x\cos^2x\,dx = \dfrac{\pi}{16}\approx 0.1963\]
Avanzado · 05 ∫ tan⁴(x) dx — reducción recursiva de índice ›
Calcular: \(\displaystyle\int\tan^4x\,dx\)
// reducción: tan⁴ → tan² → tan⁰
separar: tan⁴ = tan²·tan²
\(\int\tan^2x(\sec^2x-1)\,dx\)
distribuir
\(\int\tan^2x\sec^2x\,dx - \int\tan^2x\,dx\)
primera integral: u=tan(x)
\(\int u^2\,du = \dfrac{u^3}{3}=\dfrac{\tan^3x}{3}\)
segunda integral: reducir de nuevo
\(\int(\sec^2x-1)\,dx = \tan x - x + C\)
combinar resultados
\(\dfrac{\tan^3x}{3} - \tan x + x + C\)
\[\int\tan^4x\,dx = \dfrac{\tan^3x}{3}-\tan x+x+C\]
Avanzado · 06 ∫ sin⁴(x) dx — doble reducción con ángulo doble ›
Calcular: \(\displaystyle\int\sin^4x\,dx\)
// reducción en dos pasos
primera reducción
\(\sin^4x = \left(\dfrac{1-\cos 2x}{2}\right)^2 = \dfrac{1-2\cos 2x+\cos^2 2x}{4}\)
reducir cos²(2x)
\(\cos^2(2x)=\dfrac{1+\cos 4x}{2}\)
sustituir y simplificar
\(\sin^4x = \dfrac{3}{8}-\dfrac{\cos 2x}{2}+\dfrac{\cos 4x}{8}\)
integrar término a término
\(\dfrac{3x}{8}-\dfrac{\sin 2x}{4}+\dfrac{\sin 4x}{32}+C\)
\[\int\sin^4x\,dx = \dfrac{3x}{8}-\dfrac{\sin 2x}{4}+\dfrac{\sin 4x}{32}+C\]

// Multimedia — Energía de Señal de Audio
Una señal de audio digital pura es \(s(t)=A\sin(\omega t)\). La energía de la señal en un intervalo \([0,T]\) se define como \(E = \int_0^T s^2(t)\,dt\). Esto determina el volumen percibido (RMS — Root Mean Square).
Multimedia Energía de señal sinusoidal en un período ›
Una señal de audio es \(s(t)=3\sin(2\pi\cdot 440\cdot t)\) (nota La, 440 Hz). Calcular la energía en un período \(T=\frac{1}{440}\) s.
// integrar s²(t)
plantear la integral
\(E=\int_0^{1/440}9\sin^2(2\pi\cdot 440\cdot t)\,dt\)
reducir sin²
\(E=9\int_0^{1/440}\dfrac{1-\cos(2\cdot 2\pi\cdot 440\cdot t)}{2}\,dt = \dfrac{9}{2}\int_0^{1/440}(1-\cos(1760\pi t))\,dt\)
integrar
\(\dfrac{9}{2}\left[t-\dfrac{\sin(1760\pi t)}{1760\pi}\right]_0^{1/440}\)
evaluar: sin(4π)=0
\(\dfrac{9}{2}\cdot\dfrac{1}{440} = \dfrac{9}{880}\) Joules (por período)
Energía por período: \(E = \dfrac{A^2 T}{2} = \dfrac{9}{880} \approx 0.01023\) J
🎮
// aplicación en audio digitalEl valor RMS (Vrms = A/√2 = 3/√2 ≈ 2.12) es lo que mide el medidor de volumen de un DAW (Digital Audio Workstation). La integral de sin² es la base matemática del nivel en dB: dB = 20·log₁₀(Vrms). El resultado E=A²T/2 muestra que la energía de una sinusoide pura es siempre la mitad de su amplitud al cuadrado por el período, sin importar la frecuencia.
// Industrial — Eficiencia de Producción Cíclica
Una línea de producción opera en ciclos. La tasa de producción efectiva varía con el tiempo debido a la fatiga operativa y el mantenimiento preventivo.
Industrial Producción acumulada con eficiencia oscilante ›
La tasa de producción de una planta (unidades/hora) es: \[P(t) = 500\cos^2\!\left(\frac{\pi t}{8}\right) + 200\] donde t es la hora del turno de 8 horas. Calcular la producción total del turno y la tasa promedio horaria.
// integral de P(t) en [0,8]
parte constante
\(\int_0^8 200\,dt = 1600\) unidades
reducir cos²(πt/8)
\(500\int_0^8\dfrac{1+\cos(\pi t/4)}{2}\,dt = 250\left[t+\dfrac{4\sin(\pi t/4)}{\pi}\right]_0^8\)
evaluar: sin(2π)=sin(0)=0
\(250\times 8 = 2000\) unidades
producción total
\(Q_{total} = 1600 + 2000 = 3600\) unidades
tasa promedio
\(\bar{P} = 3600/8 = 450\) unidades/hora
\[Q_{total} = 3600 \text{ unidades} \qquad \bar{P} = 450 \text{ u/h}\]
⚙️
// decisión operacionalLa producción oscila entre 200 u/h (mínimo al final del turno) y 700 u/h (inicio). El promedio de 450 u/h informa la capacidad planificada del inventario. Si el cliente pide 4000 unidades/turno, la planta tiene un déficit del 11% y se necesita reducir el coeficiente de variación en el proceso.
// Ciencia de Datos — Ortogonalidad de Funciones de Base
En ML y análisis espectral, las funciones sinusoidales forman una base ortogonal: \(\int_{-\pi}^{\pi}\sin(mx)\cos(nx)\,dx=0\) para todos m,n. Esta propiedad hace posible la Transformada de Fourier y los Kernels trigonométricos.
Datos Verificar ortogonalidad y calcular coeficientes de Fourier ›
(a) Verificar que \(\int_{-\pi}^{\pi}\sin^2(x)\,dx = \pi\).
(b) Calcular el coeficiente \(b_1 = \dfrac{1}{\pi}\int_{-\pi}^{\pi}f(x)\sin(x)\,dx\) para \(f(x)=x\) en \([-\pi,\pi]\).
// parte (a) — norma de la función base
reducir sin²(x)
\(\int_{-\pi}^{\pi}\dfrac{1-\cos(2x)}{2}\,dx = \dfrac{1}{2}\left[x-\dfrac{\sin(2x)}{2}\right]_{-\pi}^{\pi}\)
evaluar
sin(2π)=sin(−2π)=0 → \(\dfrac{1}{2}(\pi-(-\pi))=\dfrac{2\pi}{2}=\pi\) ✓
// parte (b) — coeficiente de Fourier b₁
integrar por partes
\(\int_{-\pi}^{\pi}x\sin(x)\,dx\): sea u=x, dv=sin(x)dx → \([-x\cos x]_{-\pi}^{\pi}+\int_{-\pi}^{\pi}\cos x\,dx\)
evaluar
\((-\pi(-1)-(\pi(-1)))+[\sin x]_{-\pi}^{\pi} = 2\pi + 0 = 2\pi\)
calcular b₁
\(b_1 = \dfrac{1}{\pi}\cdot 2\pi = 2\)
\[b_1 = 2 \quad\Rightarrow\quad f(x)\approx 2\sin(x)-\sin(2x)+\frac{2}{3}\sin(3x)-\cdots\]
📊
// conexión con MLLa ortogonalidad implica que cada coeficiente de Fourier b_n se calcula independientemente de los demás — esto hace que la Transformada de Fourier sea equivalente a una regresión lineal con base ortogonal, sin multicolinealidad. En series de tiempo (ARIMA, Prophet), los componentes de estacionalidad son exactamente estos coeficientes.
// Software — Iluminación en Gráficos 3D (BRDF)
En rendering de videojuegos, la BRDF lambertiana integra la irradiancia sobre un hemisferio. El modelo de iluminación difusa involucra \(\int\cos(\theta)\,d\Omega\) sobre el hemisferio, que se descompone en integrales de potencias trigonométricas.
Software Integral de iluminación difusa sobre hemisferio ›
La irradiancia difusa total de una superficie con albedo \(\rho\) expuesta a luz uniforme \(L\) es:
\[E = \rho L\int_0^{2\pi}\int_0^{\pi/2}\cos(\theta)\sin(\theta)\,d\theta\,d\phi\]Calcular E en función de ρ y L.
// integral sobre esfera
separar integrales
\(E = \rho L\cdot\int_0^{2\pi}d\phi\cdot\int_0^{\pi/2}\cos\theta\sin\theta\,d\theta\)
integral en φ
\(\int_0^{2\pi}d\phi = 2\pi\)
integral en θ: usar sin·cos = sin(2θ)/2
\(\int_0^{\pi/2}\dfrac{\sin(2\theta)}{2}\,d\theta = \dfrac{1}{2}\left[-\dfrac{\cos(2\theta)}{2}\right]_0^{\pi/2}=\dfrac{1}{2}\cdot 1=\dfrac{1}{2}\)
combinar
\(E = \rho L\cdot 2\pi\cdot\dfrac{1}{2}=\pi\rho L\)
\[E = \pi\rho L\]
💻
// en el shaderEn GLSL/HLSL, este resultado se usa como: irradiance = PI * albedo * light_color. El factor π es exactamente el que diferencia los modelos "radiometrically correct" de los modelos ad-hoc. OpenGL y Vulkan asumen esta integral resuelta cuando calculan el componente difuso de un material PBR (Physically Based Rendering).
// Sistemas — Potencia Promedio de Señal en Red
En telecomunicaciones, la potencia promedio de una señal portadora \(s(t)=A\cos(\omega_c t+\phi)\) determina el ancho de banda necesario y el ratio señal-ruido (SNR) en el canal.
Sistemas Potencia promedio de señal portadora modulada ›
Una señal modulada AM es \(s(t)=(1+0.5\cos(2\pi\cdot 1000\cdot t))\cos(2\pi\cdot 10^6\cdot t)\). Calcular la potencia promedio \(P=\dfrac{1}{T}\int_0^T s^2(t)\,dt\) donde T es el período de la portadora.
// expandir y aplicar propiedad de ortogonalidad
expandir s²(t)
\(s^2=(1+0.5\cos\omega_m t)^2\cos^2\omega_c t = (1+\cos\omega_m t+0.25\cos^2\omega_m t)\cos^2\omega_c t\)
propiedad: promedio de cos² = 1/2
Los términos cruzados cos(ω_m t)·cos²(ω_c t) promedian a 0 (frecuencias distintas)
calcular cada término
\(P = \dfrac{1}{2}\cdot 1 + \dfrac{0.25}{2}\cdot\dfrac{1}{2} = 0.5 + 0.0625 = 0.5625\) W
\[P = \dfrac{A^2}{2}\left(1+\dfrac{m^2}{2}\right)=\dfrac{1}{2}\left(1+\dfrac{0.25}{2}\right)=0.5625\text{ W}\]
🔗
// en sistemas de comunicaciónLa fórmula P=A²/2·(1+m²/2) muestra que la potencia total de una señal AM se distribuye: 1/(1+m²/2) en la portadora y m²/2/(1+m²/2) en las bandas laterales. Con m=0.5, solo el 11% de la potencia transmitida contiene información útil. Esto explica por qué AM es ineficiente comparado con SSB (Single Sideband) donde la integral daría P=A²m²/4.
// módulo 05 — costos marginales

Estimación de Costos Marginales

Relación fundamental: si conocemos CM(q), la integral nos da el costo total. Aplicable a producción industrial, costo computacional de algoritmos y uso de recursos en infraestructura.

\[C(q) = C_0 + \int_0^q CM(t)\,dt\]

donde \(C_0\) = costos fijos, \(CM(t)\) = costo de producir la unidad t-ésima

// ¿Por qué CM es trigonométrico?
  • ▸Multimedia: Costo de renderizado oscila con la complejidad de escenas (ciclos día/noche, efectos)
  • ▸Industrial: Fatiga de maquinaria, mantenimiento preventivo cíclico
  • ▸Software: Costo de cómputo (cloud $) por ciclo de procesamiento
  • ▸Sistemas: Costo de ancho de banda por carga oscilante en servidores
// valores promedio de largo plazo

El promedio de \(\sin^2\) y \(\cos^2\) sobre un ciclo completo es siempre ½.

\[\lim_{q\to\infty}\frac{C(q)}{q} = A + \frac{B}{2}\]

para CM(q) = A + B·sin²(ωq)

Industrial + Datos CM-1: Costo total desde costo marginal oscilatorio ›
El costo marginal de una planta (miles $/tonelada) es:
\[CM(q) = 200 + 60\sin^2\!\left(\frac{\pi q}{100}\right)\]con costos fijos C₀ = $800k. Calcular C(100) y C(200), y el costo medio de largo plazo.
// C(100)
parte constante
\(\int_0^{100}200\,dq = 20{,}000\)
reducir sin²(πq/100)
\(60\int_0^{100}\dfrac{1-\cos(2\pi q/100)}{2}\,dq = 30\left[q-\dfrac{100}{2\pi}\sin\!\left(\dfrac{2\pi q}{100}\right)\right]_0^{100}\)
evaluar: sin(2π)=0
\(30\times 100 = 3{,}000\)
C(100)
\(800 + 20{,}000 + 3{,}000 = 23{,}800\) miles de pesos
\[C(100) = \$23{,}800{,}000 \quad C(200) = \$46{,}600{,}000 \quad \overline{CM}_{LP} = \$230{,}000/\text{ton}\]
⚙️
// para IndustrialEl costo medio de LP = 200 + 60/2 = $230k/ton es el precio mínimo de equilibrio. Si el precio de mercado cae por debajo, el producto no es rentable.
📊
// para Ciencia de DatosEsta misma estructura aparece en el costo de entrenamiento de un modelo ML por época: CM(q) oscila con los mini-batches. Integrar da el costo computacional total del entrenamiento.
Software + Sistemas CM-2: Costo de infraestructura cloud con carga oscilante ›
El costo por hora de una plataforma cloud (USD/hora) varía con la carga de usuarios según:
\[CM(t) = 50 + 30\cos^2\!\left(\frac{\pi t}{12}\right)\] donde t es la hora del día (0 a 24). Calcular el costo total diario.
// integral en [0,24]
parte constante
\(\int_0^{24}50\,dt = 1{,}200\) USD
reducir cos²(πt/12)
\(30\int_0^{24}\dfrac{1+\cos(\pi t/6)}{2}\,dt = 15\left[t+\dfrac{6\sin(\pi t/6)}{\pi}\right]_0^{24}\)
evaluar: sin(4π)=0
\(15\times 24 = 360\) USD
costo total
\(1{,}200 + 360 = 1{,}560\) USD/día
\[\text{Costo diario} = \$1{,}560 \quad \Rightarrow \quad \text{Mensual} \approx \$46{,}800\]
💻
// decisión de arquitecturaEl costo promedio es $65/h pero pico a $80/h y baja a $50/h. Con auto-scaling, si apagamos instancias en horas valle (carga <50%), ahorramos hasta $360/día. El análisis de la integral da el presupuesto base antes de optimizaciones.

Comportamiento Asintótico de Tendencias

¿Qué pasa cuando la producción, los datos o las peticiones crecen indefinidamente? El análisis asintótico responde si los costos convergen o divergen.

// caso 1 — costo medio convergente
\[\lim_{q\to\infty}\frac{C(q)}{q} = A+\frac{B}{2}\]

Para \(CM(q)=A+B\sin^2(\omega q)\): el promedio de sin² = ½ en cualquier número entero de períodos. El costo medio de LP es constante. No hay economías ni deseconomías de escala.

// caso 2 — oscilaciones amortiguadas
\[CM(q) = \frac{A\sin^2(\omega q)}{q}\]

El CM decrece con q (economías de escala). La integral diverge logarítmicamente: el costo total crece pero cada unidad adicional cuesta menos → rendimientos crecientes.

Asintótico AS-1: Umbral de precio de equilibrio de largo plazo ›
Una empresa de software tiene costo marginal de cómputo (USD/1000 requests):
\[CM(q) = 120 + 80\cos^2\!\left(\frac{\pi q}{500}\right)\] Determinar el precio mínimo de equilibrio de largo plazo.
// valor promedio asintótico
promedio asintótico de cos²
Sobre muchos ciclos completos: \(\langle\cos^2(\omega q)\rangle = \frac{1}{2}\)
calcular costo medio LP
\(\lim_{q\to\infty}\dfrac{C(q)}{q} = 120 + 80\cdot\dfrac{1}{2} = 120 + 40 = 160\)
\[\text{Precio mínimo} = \$160 \text{ USD por 1000 requests}\]
💻
// pricing de SaaSFijar el precio a $160/1k req cubre exactamente los costos en el largo plazo. Un precio de $200/1k da un margen del 25%. Si un competidor ofrece $140/1k, están operando por debajo del costo y su modelo no es sostenible — datos de negocio derivados de una integral.
